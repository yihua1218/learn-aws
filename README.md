1. Frontend 與 Backend 的資料交換方式，未來可以朝使用 GraphQL 或 AWS AppSync 方向前進
2. Frontend WebApp 應該要朝 Progressive Web App 方向前進
3. Frontend 若要開發 Quick Install App 可以使用 AR.js 開發影像辨識及實境擴增，來協助使用者安裝設定裝置
